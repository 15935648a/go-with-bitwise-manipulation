# [BASIC] 題目 01：判斷奇偶性

## 題目描述

實作函式 `int is_odd(int n)`，在**不使用 `%`、`/`、`if/else`** 的情況下，
判斷整數 `n` 是否為奇數。

- 奇數回傳 `1`
- 偶數回傳 `0`

---

## 函式簽名

```c
int is_odd(int n);
```

---

## 限制

| 項目 | 規定 |
|------|------|
| 禁止使用 | `%`、`/`、`if`、`else`、`? :` 三元運算子 |
| 允許使用 | `&`、`|`、`^`、`~`、`<<`、`>>` |
| 輸入範圍 | `-2147483648 ≤ n ≤ 2147483647`（32-bit int）|

---

## 輸入格式

每行一個整數 `n`，讀到 EOF 為止。

## 輸出格式

每行輸出對應的結果 `0` 或 `1`。

---

## 範例

| 輸入 | 輸出 | 說明 |
|------|------|------|
| `3`  | `1`  | 3 是奇數 |
| `4`  | `0`  | 4 是偶數 |
| `0`  | `0`  | 0 是偶數 |
| `-7` | `1`  | 負奇數也適用 |
| `-8` | `0`  | 負偶數也適用 |

---
