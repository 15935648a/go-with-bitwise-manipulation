# [ADVANCED] 題目：找出唯一不重複的數

給定一個整數陣列，其中每個數字都出現**恰好兩次**，只有**一個數字出現一次**。
實作函式找出那個唯一出現一次的數字。

## 函式簽名

```c
int find_unique(int *arr, int n);
```

## 測試案例

```
arr = [2, 3, 2, 4, 4],       n = 5  → 3
arr = [1, 1, 7],              n = 3  → 7
arr = [9],                    n = 1  → 9
arr = [-1, 3, 3, -1, 5],     n = 5  → 5
arr = [0, 1, 0],              n = 3  → 1
```

## 限制

- 禁止使用 `if/else`
- 禁止使用排序
- 禁止使用額外的 hash table 或陣列（O(1) 空間）
- 時間複雜度必須為 O(n)
- 必須使用位元運算完成

## 提示

XOR 有兩個關鍵性質，把它們組合起來：

```
性質 1：a ^ a = 0       （相同的數 XOR 消去）
性質 2：a ^ 0 = a       （任何數 XOR 0 還是自己）
性質 3：XOR 滿足交換律與結合律
```

試著對整個陣列做一次 XOR...

## 實際應用場景

| 場景 | 用途 |
|------|------|
| **Zobrist Hashing** | 圍棋/西洋棋引擎用 XOR 增量更新盤面 hash，O(1) 落子更新 |
| **RAID 同位元檢查** | RAID-5 用 XOR 計算 parity block，一顆硬碟壞掉可還原 |
| **網路封包校驗** | 簡易 checksum/CRC 利用 XOR 偵測傳輸錯誤 |
| **密碼學 OTP** | One-Time Pad 加密：`ciphertext = plaintext ^ key` |
| **記憶體 XOR Linked List** | 用 `prev ^ next` 一個欄位同時儲存雙向指標 |
