# [BASIC] 題目：取得第 k 位元

實作函式 `int get_bit(int n, int k)`，回傳整數 `n` 的第 `k` 個位元值（0 或 1）。
位元編號從 **0** 開始，0 為最低位（LSB）。

## 函式簽名

```c
int get_bit(int n, int k);
```

## 測試案例

```
get_bit(0b1010, 1) → 1   // 0b1010 的第 1 位是 1
get_bit(0b1010, 0) → 0   // 0b1010 的第 0 位是 0
get_bit(0b1010, 3) → 1   // 0b1010 的第 3 位是 1
get_bit(0b1010, 2) → 0   // 0b1010 的第 2 位是 0
get_bit(0,      0) → 0
get_bit(-1,     7) → 1   // -1 = 0xFFFFFFFF，所有位元皆為 1
```

## 輸入格式

每行兩個整數 `n k`，直到 EOF。

## 輸出格式

每行輸出 `n` 的第 `k` 位元值（0 或 1）。

## 限制

- 禁止使用 `if/else`
- 禁止使用 `%`、`/`
- `0 <= k <= 30`
