# [INTERMEDIATE] 題目 06：無分支絕對值（Branchless Absolute Value）

## 題目描述

實作函式 `int branchless_abs(int n)`，在**不使用任何分支**的情況下，回傳 `n` 的絕對值。

**禁止使用 `if/else`、三元運算子 `? :`、或標準庫 `abs()`**，必須純位元運算實作。

---

## 函式簽名

```c
int branchless_abs(int n);
```

---

## 測試案例

| 輸入          | 輸出         | 說明                        |
|---------------|--------------|-----------------------------|
| `5`           | `5`          | 正數不變                    |
| `-5`          | `5`          | 負數取反                    |
| `0`           | `0`          | 零保持零                    |
| `-1`          | `1`          | 最小負數之一                |
| `2147483647`  | `2147483647` | INT_MAX，不需轉換           |
| `-2147483647` | `2147483647` | INT_MIN+1，安全邊界         |
| `-100`        | `100`        | 一般負數                    |
| `1`           | `1`          | 最小正數                    |

> ⚠️ **INT_MIN（-2147483648）** 的絕對值在 32-bit int 下無法表示（溢位），屬未定義行為，不列入測試。

---

## 提示

思考 `n >> 31` 對正數和負數分別產生什麼值：

- 正數：`n >> 31` → `0x00000000`（算術右移，符號位補 0）
- 負數：`n >> 31` → `0xFFFFFFFF`（算術右移，符號位補 1，即 `-1`）

利用這個 mask，搭配 XOR 與減法，可以在不分支的情況下實現取絕對值。

---

## 限制

| 項目 | 規定 |
|------|------|
| 禁止使用 | `if/else`、`? :`（三元運算子）、`abs()`、`fabs()` |
| 禁止使用 | 乘法、除法 |
| 允許使用 | `&`、`\|`、`^`、`~`、`<<`、`>>`、`+`、`-` |
| 輸入範圍 | `-2147483647 ≤ n ≤ 2147483647`（排除 INT_MIN） |

---

## 實際應用場景

本題技巧在以下領域廣泛應用：

- **CUDA kernel**：數萬個 thread 同時計算 `abs()`，branchless 可消除 warp divergence，大幅提升 GPU 吞吐量
- **DSP 信號處理**：音訊振幅、AGC（自動增益控制）需大量絕對值運算，branchless 可利用 SIMD 向量化（`_mm_abs_epi32`）
- **影像處理**：Sobel / Prewitt filter 計算像素梯度差時需要 `abs()`，GPU shader 常用此技巧
- **嵌入式系統**：無分支預測能力的低功耗 MCU（如 ARM Cortex-M0），避免分支可減少 pipeline flush
- **密碼學**：側通道攻擊防禦中，constant-time 操作要求禁止條件分支
